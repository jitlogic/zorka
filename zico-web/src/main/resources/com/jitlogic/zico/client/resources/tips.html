Welcome !

Zorka Intranet Collector collects performance data from Zorka agents across the network. You can view details about execution of your application here.

Tips

  • to display trace list from host of interest, double click host from the list from left panel;
  • to display trace details double click double click on specific trace in trace list;
  • you can search traces by regexp by adding tilde ('~') at the beginning of search phrase
  • or you can search specific trace attributes using @ATTR=text or @ATTR=~regex searcg phrases;
  • setting Start time and End time will make your searches faster;
